About the Student Retention Add-in

Version 1.0

Overview

This add-in is designed to streamline the process of tracking and managing student retention by integrating key data directly within your Excel workbook.

The "Retention" Ribbon

After installation, a new "Retention" tab will appear on the Excel ribbon. This tab contains all the tools you need to interact with your student data.

All features of this add-in are accessible from this central location on the ribbon.

The "Show Details" Task Pane

This is the main interface of the add-in, providing a comprehensive, at-a-glance view of the selected student. It opens when you click the "Show Details" button on the ribbon.

Student Details View

When you select a student's row in your worksheet, this view automatically populates with their information:

  • Identity: A color-coded avatar (based on gender, if available), full name, assigned advisor, student ID, and last LDA.
  • Key Stats: "Days Out" and "Grade" are displayed prominently with color-coding for immediate status assessment (e.g., green for good, red for at-risk).
  • Contact Info: Phone numbers and email addresses are listed. Clicking on any of these will instantly copy the information to your clipboard.

Student History View

This view, accessible via a tab within the task pane, provides a complete interaction history for the selected student, pulled from the "Student History" sheet.

  • View History: Displays all previously logged comments and outreach notes in reverse chronological order.
  • Add New Comment: Allows you to quickly add new comments or notes. These are saved to the "Student History" sheet with the current timestamp and your name.

Import Data Button

This feature opens a dialog to import data from .csv or .xlsx files. The add-in intelligently detects the file's content to offer the appropriate actions.

Update Master List

  • Functionality: Updates existing student records and adds new students from the import file.
  • Process: New students are inserted at the top of the "Master List" sheet (below the header) and highlighted in light blue for easy identification. Existing students are updated in place.
  • Requirement: The import file must contain a column for the student's name and student number/identifier.

Update Grades

  • Functionality: Updates grades, gradebook links, and assignment statuses for students already present in your "Master List".
  • Process: This function will not add new students. It only modifies the records of existing students.
  • Requirement: The import file must contain columns for student name, course, and current score/grade.

Contacted Button

A simple but effective tool for visual tracking. When you select a student's row and click the "Contacted" button, it toggles a yellow highlight across the main columns of that row.

This provides a quick, visual way to see which students you've already reached out to during a session. Clicking the button again removes the highlight.
