About the Student Retention Add-in

Version 1.0

Overview

This add-in is designed to streamline the process of tracking and managing student retention by integrating key data directly within your Excel workbook.

The "Retention" Ribbon

After installation, a new "Retention" tab will appear on the Excel ribbon. This tab contains all the tools you need to interact with your student data.

All features of this add-in are accessible from this central location on the ribbon.

Initial Setup

For the add-in to function correctly, your workbook must contain two specific sheets:

  • Master List: This sheet should contain your main list of students, with a header row (row 1) that includes columns like StudentName, StudentNumber, Grade, etc.
  • Student History: This sheet is used to log all comments and outreach. It needs a header row with at least StudentNumber, Comment, and Timestamp columns.

First Use

The first time you open the "Show Details" task pane, a welcome message will appear asking for your first and last name. This name will be used to automatically sign any comments you add to a student's history.

Show Details

This is the main interface of the add-in. When you select a student's row, this pane automatically populates with their information, including contact details, key stats, and a full interaction history.

Import Data

This feature opens a dialog to import data from .csv or .xlsx files. The add-in intelligently detects the file's content to offer the appropriate actions: "Update Master List" or "Update Grades".

Create LDA

Generates a new sheet named with the current date. This sheet contains two tables:

  • A list of students who are "at-risk" based on the "Days Out" filter you set in the Settings.
  • A list of currently failing students who have been active in the last 4 days.

Contacted

A simple but effective tool for visual tracking. When you select a student's row and click this button, it toggles a yellow highlight across the main columns of that row.

Transfer Data

Copies key data from the active sheet (Student Name, Gradebook link, Days Out, etc.) to your clipboard in JSON format, ready to be pasted into other applications.

Settings Pane

Clicking the "Settings" button on the ribbon opens a task pane where you can customize the add-in's behavior.

  • User Profile: Set the first and last name used to sign your comments.
  • LDA Report Settings:
    • Set the "Days Out" threshold for the at-risk student list.
    • Toggle whether to include the "Failing Students" list in the report.
    • Choose whether to treat empty cells in an imported grade file as a grade of 0.
  • Column Management: Drag and drop column names to customize which columns appear in the generated LDA report and in what order.
