About the Student Retention Add-in

Version 1.5.0.0

This add-in is designed to streamline the process of tracking and managing student retention by integrating key data directly within your Excel workbook.

How It Works

The add-in adds a new "Retention" tab to the Excel ribbon with several tools to help you manage your student data efficiently.

Ribbon Buttons

  • Import Data: Opens a dialog to import data from .csv or .xlsx files. The add-in automatically detects the file type to offer relevant actions:
    • Update Master List: Updates existing student records in your "Master List" sheet and adds new students to the top of the sheet, highlighted for visibility.
    • Update Grades: Updates grades and gradebook links for students already present in your "Master List".
  • Contacted: Toggles a yellow highlight on the currently selected student row to visually mark that contact has been made.
  • Show Details: Opens the main task pane where you can view detailed student information and their history.
  • About: Opens this informational pane.

The "Show Details" Task Pane

This is the main interface of the add-in, providing a comprehensive view of the selected student.

Student Details Tab

When you select a row in your worksheet, this tab automatically populates with that student's information, including:

  • A color-coded avatar and name.
  • Key stats like "Days Out" and "Grade", which are color-coded for at-a-glance status checks.
  • Contact information (phone numbers and emails) that can be copied to your clipboard with a single click.

Student History Tab

This tab provides a complete interaction history for the selected student, pulled from the "Student History" sheet.

  • View History: Displays all previously logged comments and outreach notes in chronological order.
  • Add New Comment: Allows you to quickly add new comments or notes for the selected student, which are then saved to the "Student History" sheet with a timestamp.
